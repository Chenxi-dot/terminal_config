# 终端轻量化部署指南 (Beautiful Edition)

本文档介绍了一套**极简、纯净、高性能**且**美观**的终端配置方案。

**核心理念**：
*   **美观优先**：采用 **Nord (Gogh)** 配色方案，搭配紫色磨砂玻璃背景，提供沉浸式视觉体验。
*   **极速启动**：使用 Rust 编写的 **Starship** 作为提示符，启动速度极快。
*   **纯净环境**：仅手动挂载最核心的“语法高亮”和“自动建议”插件，不污染系统。
*   **全自动与容错**：脚本内置网络重试、原子写入、自动备份和自检机制，确保部署过程万无一失。

---

## 1. 核心组件

1.  **WezTerm**: GPU 加速的终端模拟器。
    -   **主题**: Nord (Gogh) + 紫色磨砂背景 (#301934, Blur 25)。
    -   **字体**: `JetBrainsMono Nerd Font` (自动下载，完美支持所有图标)。
    -   **特性**: 智能分屏 (Leader: Ctrl+A), 底部标签栏, 窗口无边框设计。

2.  **Zsh**: 默认 Shell。
    -   **插件**: `zsh-syntax-highlighting` (语法高亮), `zsh-autosuggestions` (自动建议)。
    -   **功能**: 启动问候语, `shutdown` 别名, 集成 `fzf` 和 `zoxide`。

3.  **Starship**: 极速 Shell 提示符。
    -   **风格**: **Beautiful Edition**。
    -   **信息**: 右侧状态栏显示详细信息 (时间, 电池, 内存等)，自定义 Git 分支图标 (△)。

4.  **现代化工具**:
    -   `zoxide`: 智能目录跳转 (`z` 替代 `cd`)。
    -   `fzf`: 模糊搜索 (集成到 Zsh)。
    -   `eza`: `ls` 的现代化替代品 (带图标)。
    -   `bat`: `cat` 的现代化替代品 (带语法高亮)。

## 2. 快速开始

### Mac / Linux 用户
1.  打开终端，进入脚本目录。
2.  运行：
    ```bash
    chmod +x setup_terminal.sh
    ./setup_terminal.sh
    ```
3.  根据提示选择是否开启代理加速下载。
4.  重启终端即可。

---

## 3. 目录结构 (XDG 标准)

脚本会自动清理旧的 Home 目录配置文件，并将新配置放置在标准位置，保持 Home 目录整洁：

-   `~/.config/wezterm/` (WezTerm 配置)
-   `~/.config/starship.toml` (Starship 配置)
-   `~/.zshrc` (Zsh 主配置)
-   `~/.local/bin/` (工具安装位置)

---

## 4. 常见问题

*   **字体乱码？**
    脚本会自动下载并安装 `MesloLGS NF` 字体。如果仍有乱码，请手动在 Terminal 设置中确认字体是否生效，或重启电脑。

*   **下载慢？**
    脚本启动时会询问是否配置代理。如果下载速度慢，请选择 `y` 并输入你的代理地址 (如 `http://127.0.0.1:7890`，具体需要依据你的 `vpn` 进行调整)
